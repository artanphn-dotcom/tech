Cloud Storage

gcloud storage ls

gcloud storage ls gs://[BUCKET_NAME]

What it does: Lists the contents of a Cloud Storage bucket.

gcloud storage cp

gcloud storage cp [LOCAL_FILE] gs://[BUCKET_NAME]/

What it does: Copies files between your local machine and Cloud Storage.

gcloud storage rm

gcloud storage rm gs://[BUCKET_NAME]/[OBJECT_NAME]

What it does: Removes objects from a Cloud Storage bucket.

gcloud storage buckets describe

gcloud storage buckets describe gs://[BUCKET_NAME]

What it does: Displays detailed information about a Cloud Storage bucket.

gcloud storage objects list

gcloud storage objects list gs://[BUCKET_NAME]

What it does: Lists objects within a Cloud Storage bucket.

gcloud storage buckets create

gcloud storage buckets create gs://my-unique-bucket-name --project=[PROJECT_ID] --location=us-central1

What it does: Creates a new Cloud Storage bucket.

gcloud storage buckets update

gcloud storage buckets update gs://[BUCKET_NAME] --default-storage-class=NEARLINE

What it does: Updates properties of a Cloud Storage bucket.

gcloud storage buckets delete

gcloud storage buckets delete gs://[BUCKET_NAME]

What it does: Deletes a Cloud Storage bucket (must be empty).

gcloud storage acl get

gcloud storage acl get gs://[BUCKET_NAME]

What it does: Displays the Access Control List (ACL) for a bucket.

gcloud storage acl set

gcloud storage acl set public-read gs://[BUCKET_NAME]

What it does: Sets predefined ACLs for a bucket, e.g., making it publicly readable.

gcloud storage objects describe

gcloud storage objects describe gs://[BUCKET_NAME]/[OBJECT_NAME]

What it does: Displays detailed information about a specific object.

gcloud storage objects update

gcloud storage objects update gs://[BUCKET_NAME]/[OBJECT_NAME] --content-type=text/html

What it does: Updates metadata for a Cloud Storage object.

gcloud storage compose

gcloud storage compose gs://[BUCKET]/object1.txt gs://[BUCKET]/object2.txt gs://[BUCKET]/composed_object.txt

What it does: Concatenates multiple objects into a single new object.

gcloud storage cp --recursive

gcloud storage cp --recursive gs://[SOURCE_BUCKET]/folder/ gs://[DESTINATION_BUCKET]/

What it does: Copies an entire directory recursively between buckets or local filesystem.

gcloud storage du

gcloud storage du gs://[BUCKET_NAME]

What it does: Reports disk usage of buckets and objects.

gcloud storage events list

gcloud storage events list gs://[BUCKET_NAME]

What it does: Lists Cloud Storage event notifications.

gcloud storage rsync

gcloud storage rsync -r ./local_dir gs://[BUCKET_NAME]/remote_dir

What it does: Synchronizes content between a local directory and a bucket, or between two buckets.

gcloud storage set-default-event-based-hold

gcloud storage set-default-event-based-hold gs://[BUCKET_NAME] --enable

What it does: Enables default event-based hold on new objects in a bucket.

gcloud storage buckets add-iam-policy-binding

gcloud storage buckets add-iam-policy-binding gs://[BUCKET_NAME] --member='user:user@example.com' --role='roles/storage.objectViewer'

What it does: Grants IAM permissions to a bucket.

gcloud storage buckets get-iam-policy

gcloud storage buckets get-iam-policy gs://[BUCKET_NAME]

What it does: Retrieves the IAM policy for a bucket.
